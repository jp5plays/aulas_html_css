ID E class

Existem duas características principais do class

  • É possível utilizar o mesmo class em vários elemntos diferentes
  • É possível utilizar várias classes no MESMO elemento

ID

Existem duas caracteríscas contrastantes do id que são contrastantes com as características do class

  • Não é possível utilizar o mesmo ID em vários elementos diferentes
  • Não é possível utilizar vários IDs no mesmo elemento

O id funciona como um cpf. Não podem existir 2 iguais e a mesma pessoa não pode ter vários CPFs
